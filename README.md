# بوتيك عهود — Boutique Ohood

موقع بوتيك أزياء نسائية محتشمة يستهدف **المملكة العربية السعودية، خصوصاً الرياض**.

## الهوية

**عهود** — اسم سعودي أنثوي يحمل معنى الوعد والالتزام.  
الشعار: **«أناقة تحتفظ بوعودها»**

- أناقة محتشمة (عبايات، جلابيات، قصات مغطاة)
- جودة تفي بوعدها
- خدمة الرياض في نفس اليوم

## المميزات

- **العربية افتراضياً** مع دعم الإنجليزية (تبديل EN / العربية)
- تخطيط RTL كامل
- كتالوج منتجات + تصفية وترتيب
- عرض سريع، حقيبة، مفضلة (تُحفظ محلياً)
- توصيل مجاني فوق ٤٠٠ ر.س
- واتساب للتنسيق
- صور محتشمة فقط

## التشغيل محلياً

```bash
cd /path/to/u
python3 -m http.server 8080
```

ثم افتحي: `http://localhost:8080`
